# Run a complete extraction for a playlist id

Run a complete extraction for a playlist id

## Usage

``` r
run_complete_extraction(
  api_key,
  yt_dlp_path,
  path,
  suffix,
  playlist_id,
  max_videos
)
```

## Arguments

- api_key:

  youtube api key

- yt_dlp_path:

  path to yt_dlp

- path:

  path to save subtitles

- suffix:

  suffix to identify the playlist

- playlist_id:

  id of the playlist

- max_videos:

  maximum amont of videos to extract

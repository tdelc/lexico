# Download subtitles from a youtube video

Download subtitles from a youtube video

## Usage

``` r
download_subtitles(video_id, out_dir, yt_dlp = "yt-dlp", force_dl = FALSE)
```

## Arguments

- video_id:

  video id

- out_dir:

  directory to save file

- yt_dlp:

  name of program to launch

- force_dl:

  overide existed vtt file
